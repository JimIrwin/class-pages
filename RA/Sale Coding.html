Code  QC if the language of the transfer includes the words "quit claim" -- these are not regular sales.
If more specific information is noticed, use a period to code it:  sale.auct (sold at auction)     sale.enf [the word "enfeoff" is in the sale language; but sale.auct for auction sale with enfeoff in the deed ]   Note, for a regular sale, various combinations of verbs may be used, e.g. in Albany:  "grant bargain and sell"   or  "sell convey release and confirm" etc.    ] 

sale.grant : for Henrico in the 1850s, many deeds simply use the word "grant" without the usual other words (bargain, sell, convey, etc.).  
sale.gw -- when the deed says "grant ...  with general warranty" (and other words like "bargain and sell" are not used.  Many deeds that "grant" land are deeds to "secure" some debt, but some are simply sales -- read through the deed to identify it for coding of the deed variable as "sale.grant" versus "secure"  .

FPnames:  name(s) & occupation(s) of the party(s) of the First Part.  With many persons, record surnames only.  If more than five surnames, simply record three surnames and add  ETC.

The coding format for names is very important -- pay close attention to the conventions we use.
Format names as Surname firstname middlename (or initial)  e.g.  Anthony susan b     King martin luther Jr  
Surnames start with a capital letter (upper case). 

Periods are not used when coding names. A comma separates people.  & joins spouses (use & for spouses, and only for spouses).

 Surnames start with a capital letter, and surnames are not repeated (instead use a comma, where a lower case letter following a comma indicates the same surname as the prior person).
Thus:  "Taylor willam (farmer) & ann"      "Obama barack & michelle, malia, sasha"         "James jesse, frank, Washington martha"     "Smith, Jones, Brant, VanEyck"   "GROUP of 12"

[Albany deeds]  Ci.AlbW1 Ci.AlbW9  Ci.AlbW12   Ci.Alb (ward not given)   Co.Alb (more specific location not given)
  T.Waterv  Waterv   T.Guild  Guild   T.Westerl    T.WTroy   T.Schen   Renss V.Renss  T.Renss   WTroy V.WTroy   
Locations outside of Albany County,    
if a city, code  Ci.xxxx   e.g.  Ci.NYC  Ci.Troy   Ci.Phila  Ci.Boston  ;
else code other counties as Co.xxxxx e.g.  Co.Renss  Co.Cayug  Co.Dutchess ; other states as S.xx  e.g.  S.MA  S.PA S.OH ; other nations as N.xxxxxx   e.g.  N.Canada   N.England  N.Mexico
[Hco and Hust deeds]   Co.Hco Pa.Hco  Ci.Rich  

* use a prefix only as the prefix is used in the deed (V.WTroy for "Village of West Troy"    WTroy for  "West Troy"     Hco for "Henrico"  Pa.Hco for "Parish of Henrico"   Co.Hco for "County of Henrico").

 fem  (1 if a female is a party to the deed, and not as a W; 0 otherwise    ? for unsure)

 illit (illiteracy?).  Code of 0 denotes signatures and not marks by all the parties of the first part. If  1P person(s) marked, then code the "type" of 1P person who used a "mark" (instead of signing the deed): W for a wife marked, H for a husband,  m for a male marked, etc.  -9 neither mark nor signature for one of more of the parties of the first part.

Type (of property) -- be aggressive with abbreviations, to signal what is written :  "p p land"  for piece or parcel of land   "l tr land"   for lot or tract of land    "l p ground"  for lot or parcel of ground. Etc.  When multi pieces are involved, start with a number ("2 l p"   for two lots or pieces).  Precision is not important for this variable, so do not spend time fussing with it.

acres (for land when size is given in acres)

some more discussion

Deed -- key issue is sale or other (other deeds include: QC quit claim, mortgage, release, secure, bond,  ).   When unsure it is best to code the deed as a possible sale -- code deed as sale?  and highlight it for Jim's attention.  

sale.enf  :  "enfeoff"   The "deed" variable is coded sale.enf  if the word "enfeoff" is used in the language that describes the transfer of the land (for example, in Albany deeds "by these presents doth grant, bargain, sell, alien, remise, release convey, assure enfeoff and confirm."   Code simply as "sale" when enfeoff is not there (such as "grant bargain sell alien convey and confirm" ).  Be alert for the word "quit claim"  -- code that as QC and leave it for Jim to determine whether it is a sale or not.

Type : type of land as stated in source, for example "lot, piece, or parcel" (l p or p) or  "tract of land" (tract).  Use abbreviations ("l p or p" for "lot, piece, or parcel"    "tract" for "tract of land").
If coding a non-sale this variable (Type) is optional,  to code if noticed ...  RE (other real estate)   /    slave(s) / PE (other personal estate) / other.

bldg :  0 no mention of buildings;           1 buildings mentioned (e.g.  house, barn, tenement, buildings).

geog: geographic details, usually blank, but if you notice other info, enter it here (detailed location, lot number, rivers nearby -- similar to a comment, but this is info about the land)


more details, if needed

Court: Hco or Hust or NYAlb or Essex  (identifies the court from which deed is taken)         DB: deed book number         Page: the page on which the item starts


xtra: 0 if deed is in the random sample; 1 if it is an extra deed (not in the random sample).  Coded by Dr Irwin only.

Deed:  sale / secure / release / other.
sale ...  of land or lot or other real estate
secure ...  deed that uses property to secure a debt (providing collateral or security for a debt).
release ...  deed that releases property that was used as security.
oth ...   (not one of the above); use a period to append more info -- oth.QC for quit claim, oth.lease for a lease


SPtype :  parties of the second part (and third part, if appropriate). Same coding as FPtype, but use a comma to separate 3P from 2P;  mm,m  indicates two men are the parties of the second part and one man is the party of the third part. HW,m would indicate husband and wife as parties of the 2nd part and one man as party of the third part.


value :  required entry for land sales, where “value” is the price; optional entry for other deeds (worth spending 10 seconds looking).  For deeds that "secure" or "release" property for debts, if you notice the money value involved then record it, but do not take time searching for that money value.  For land sales, record the price paid (usually indicated by "in consideration of  the payment of ... ") or record "not found."


bldg:  0 or 1  ; 1 if buildings are mentioned (e.g. house, tenement, barn, structures), 0 otherwise
